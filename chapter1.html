Key Definitions

Assembler

A translator that converts instruction mnemonics back into their original numeric opcodes.

Opcode

The unique numeric value that tells the processor which operation to perform.

Mnemonic

Abbreviations of symbols (e.g., mov, add) used to make numeric opcodes easier for humans to remember.

Registers

Small, temporary storage locations inside the processor used as a "scratch pad" to hold operands and results during execution.

Address Bus

A unidirectional bus used by the processor to specify the memory location it wants to read from or write to.

Data Bus

A bidirectional bus used to transfer actual data between the processor and memory or peripherals.

Control Bus

A group of lines used to synchronize operations, such as signaling whether a "read" or "write" is occurring.

Instruction Pointer (IP)

A special register that always contains the memory address of the next instruction to be executed.

Flags Register

A register where individual bits (flags) represent the status of the last mathematical or logical operation.

Key Definitions

Conditional Jump

A jump that only occurs if a specific condition (based on flags) is met (e.g., JZ jumps if Zero flag is set).

Unconditional Jump

A jump that always occurs, regardless of the flag states (e.g., JMP).

Relative Addressing

A jump target specified as a "signed displacement" (offset) from the current Instruction Pointer, rather than an absolute address.

CMP (Compare)

An instruction that subtracts the source from the destination and updates flags without changing either operand.

Two's Complement

The standard way of representing signed numbers where the MSB indicates the sign.

Signed vs Unsigned Jumps

Different jump instructions used depending on how the data is interpreted (e.g., JG for signed vs JA for unsigned).

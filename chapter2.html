Key Definitions

Direct Addressing

Accessing memory by specifying the actual memory address (or a label representing it) in the instruction.

Data Declaration

Using directives like db (define byte) or dw (define word) to reserve memory space for data.

Size Mismatch Error

An error that occurs when the size of the source operand does not match the size of the destination operand (e.g., mov al, [word_variable]).

Effective Address

The final offset of a memory location calculated by the processor after combining segments, bases, and indexes.

Label

A human-readable name that the assembler replaces with a memory address during the translation process.

Register Indirect Addressing

Accessing memory where the address is stored inside a register (e.g., mov ax, [bx]).

Segment Association

The rule that determines which segment register is used by default with a specific offset register (e.g., BX defaults to DS).

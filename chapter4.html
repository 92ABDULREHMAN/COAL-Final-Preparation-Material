Key Definitions

AND Operation

A logical operation used to mask or clear specific bits. If any bit is ANDed with 0, the result is 0.

OR Operation

A logical operation used to set specific bits. If any bit is ORed with 1, the result is 1.

XOR Operation

A logical operation used to invert or toggle specific bits. It is also a fast way to clear a register (xor ax, ax).

NOT Operation

A unary operation that inverts every bit in the operand (1 becomes 0, 0 becomes 1).

SHL (Shift Left)

Moves bits to the left, inserting 0 at the LSB. Effectively multiplies an unsigned number by 2.

SHR (Shift Right)

Moves bits to the right, inserting 0 at the MSB. Effectively divides an unsigned number by 2.

SAR (Shift Arithmetic Right)

Shifts bits right but preserves the sign bit (MSB). Used for signed division.

ROL/ROR (Rotate)

Circular shifts where the bit pushed out of one end is inserted back into the other end.

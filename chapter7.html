Key Definitions

Interrupt

A mechanism to divert the processor from its current execution to a specialized routine (ISR) to handle a specific event.

Interrupt Vector Table (IVT)

A table located at the very beginning of memory (0000:0000) containing the 4-byte addresses (CS and IP) of all 256 possible interrupts.

ISR (Interrupt Service Routine)

The code that executes when an interrupt occurs.

IRET

A special return instruction for interrupts that pops the Flags register, CS, and IP from the stack to resume the original program.

INT Instruction

A software-triggered interrupt that behaves like a far call but also saves the flags register on the stack.

Key Definitions

Video RAM

A dedicated portion of memory (starting at 0xB8000 for color) that holds the data displayed on the screen.

Attribute Byte

The second byte of every character in video memory that defines color (foreground/background) and blinking.

Memory Mapped I/O

A technique where hardware devices (like the screen) are accessed by reading/writing to specific memory addresses.

CGA/EGA/VGA

Different video standards; CS401 focuses on the 80x25 text mode (CGA).

ASCII Code

The first byte of every character in video memory representing the character itself.
